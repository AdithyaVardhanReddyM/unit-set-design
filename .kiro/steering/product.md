# Product Overview

Unit {set} is an AI-powered design-to-code platform that transforms wireframes into production-ready web applications.

## Core Features

- Secure authentication with Clerk
- Project management with dedicated canvases
- Interactive drawing and sketching tools:
  - Select, move, and resize shapes
  - Draw frames, rectangles, ellipses, lines, arrows
  - Freehand drawing with smooth rendering
  - Text with full typography controls
  - Screen shapes for AI-generated UI previews
  - Eraser tool for quick deletions
- Infinite canvas with pan and zoom:
  - Pan with middle/right mouse or Space + drag
  - Zoom with Ctrl/Cmd + wheel (zooms around cursor)
  - Hand tool with Space key (temporary pan mode)
  - Zoom to fit all shapes
- AI-powered generation of landing pages, dashboards, and components:
  - Chat with AI agent via sidebar panel
  - Persistent conversation history per screen
  - Agent writes and executes code in isolated E2B sandbox
  - Sandbox session persistence with auto-pause (15-minute timeout)
  - Live preview of generated UI with hot reload
  - Full Next.js + shadcn/ui environment
  - TypeScript validation before completion
- Visual editor with Figma-like properties panel:
  - Stroke color, width, and type (solid/dashed)
  - Corner radius (sharp/rounded)
  - Frame fill colors
  - Text typography controls (font, size, alignment)
  - Dimensions display
- Layers sidebar for shape management
- History with undo/redo (Ctrl/Cmd+Z, Ctrl/Cmd+Y)
- Copy/paste shapes (Ctrl/Cmd+C, Ctrl/Cmd+V)
- Auto-save to localStorage with Convex sync
- Real-time collaboration across canvases (planned)
- Responsive design generation

## Product Vision

Bridge the gap between design and development by enabling users to transform creative sketches into functional, interactive applications within a single canvas environment.
